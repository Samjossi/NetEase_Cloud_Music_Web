# 网易云音乐桌面版

使用PySide6为网易云音乐网页播放器创建的原生桌面应用壳。

## 功能特性

- 🎵 完整保留网页版播放器功能
- 🖥️ 原生桌面窗口体验
- 📱 支持窗口缩放和基本控制
- 🔐 登录状态持久化保存
- 🚀 简洁轻量的实现

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

#### 方法一：使用启动脚本（推荐）
```bash
python run.py
```

#### 方法二：直接运行主程序
```bash
python main.py
```

```
NetEase_Cloud_Music_Web/
```

## 技术实现

- **PySide6**: Qt6的Python绑定，用于创建原生窗口
- **QWebEngineView**: 用于嵌入和显示网页内容
- **网易云音乐播放器**: https://music.163.com/st/webplayer

## 系统要求

- Python 3.8+
- 支持的操作系统：Windows, macOS, Linux

## 注意事项

- 首次运行可能需要一些时间来初始化WebEngine
- 确保网络连接正常，需要加载网易云音乐网页
- 音频播放需要系统支持相应的音频格式
- **登录状态会自动保存**：扫码登录后，下次启动应用会自动保持登录状态
- **隐私保护**：用户数据存储在本地`web_data/`目录，该目录已加入.gitignore避免泄露

## 登录状态说明

应用会自动保存你的登录状态：
- 📱 扫码登录一次后，无需重复登录
- 💾 登录信息安全存储在本地`login_data/`目录
- 🔄 播放历史、收藏等数据由网易云音乐云端同步
- 🛡️ 精简存储：仅保存登录相关的cookies和必要数据
- 🚫 不存储音频缓存等大文件，保持轻量化
