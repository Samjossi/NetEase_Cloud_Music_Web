# 系统托盘功能实现计划

## 项目概述

为网易云音乐桌面版添加基于Linux平台的AppIndicator3技术的系统托盘功能，提供稳定、永久显示的系统托盘体验。

## 功能需求

### 核心功能
- ✅ 系统托盘图标显示
- ✅ 当前播放歌曲名称显示
- ✅ 右键菜单（显示界面、退出程序）
- ✅ 点击托盘图标显示/隐藏窗口
- ✅ 关闭窗口时最小化到托盘

### 技术要求
- 基于AppIndicator3技术实现
- 支持Linux系统桌面环境
- 提供备用方案（Qt QSystemTrayIcon）
- 稳定可靠的运行机制

## 技术实现细节

### 1. AppIndicator3集成
```python
# 检测和导入
try:
    import gi
    gi.require_version('AppIndicator3', '0.1')
    from gi.repository import AppIndicator3 as appindicator
    from gi.repository import Gtk as gtk
    APPINDICATOR_AVAILABLE = True
except (ImportError, ValueError):
    APPINDICATOR_AVAILABLE = False
```

### 3. 信号槽机制
```python
# 信号定义
show_window_requested = pyqtSignal()
exit_requested = pyqtSignal()

# 信号连接
self.tray_manager.show_window_requested.connect(self.show_window)
self.tray_manager.exit_requested.connect(self.exit_application)
```

### 4. 窗口行为修改
```python
def closeEvent(self, event):
    # 最小化到托盘逻辑
    if self.tray_manager and self.tray_manager.is_visible:
        self.hide()
        event.ignore()
        return
    # 正常退出流程
    super().closeEvent(event)
```

## 故障排除

### 常见问题
1. **托盘不显示**
   - 检查系统依赖是否安装
   - 确认桌面环境支持AppIndicator3
   - 查看日志错误信息

2. **歌曲信息不更新**
   - 检查网页是否完全加载
   - 确认JavaScript执行正常
   - 查看选择器是否匹配

3. **菜单不响应**
   - 检查信号连接是否正确
   - 确认事件处理正常
   - 查看异常日志
